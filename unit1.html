What are the basic services provided by the Internet?

1.Communication Services like Instant Messaging,Email,Video Conferencing.
2.Information Retrieval Services:
 It allows users to access remote computers and transfer files from one machine to another.It contains two protocols such as FTP(File Transfer Protocol) and SFTP(Simple File Transfer Protocol).
  FTP:It is an internet utility software which is used for uploading and downloading files.FTP enables access to remote computer folders and allows files and folders to be transferred between different computers.
  SFTP:SFTP stands for SSH File Transfer Protocol. Sometimes it is called Secure File Transfer Protocol. It is more secure than FTP. It supports the complete SSH protocol protection and authentication feature, including the      SSH key.

5.Ecommerce:E-commerce is Internet-based buying and sale of goods and services. Today, almost anything that is available in the market can be purchased through e-commerce.
6.WWW(World Wide Web):Most of us think that the Internet and WWW are the same but actually not. Internet is Infrastructure and the Web is just a service provided on the Internet. Internet is a network of networks and Web is            just information present in the network.
  a)World Wide Web, which is also known as a Web, is a collection of websites or web pages stored in web servers and connected to local computers through the internet.
  b)The World Wide Web was invented by a British scientist, Tim Berners-Lee in 1989. He was working at CERN at that time. Originally, it was developed by him to fulfill the need of automated information sharing between    scientists across the world, so that they could easily share the data,results of their experiments and studies with each other.
  c)Tim focused on three main technologies that could make computers understand each other, HTML, URL, and HTTP. So, the objective behind the invention of WWW was to combine recent computer technologies, data    networks,and hypertext into a user-friendly and effective global information system.
  d)These websites contain text pages, digital images, audios, videos, etc. Users can access the content of these sites from any part of the world over the internet using their devices such as computers, laptops, cell phones, etc.
  e)The building blocks of the Web are web pages which are formatted in HTML and connected by links called "hypertext" or hyperlinks and accessed by HTTP. These links are electronic connections that link related         information so that users can access the desired information quickly.
  f)A web page is given an online address called a Uniform Resource Locator (URL). A particular collection of web pages that belong to a specific URL is called a website. The URL consists of two parts : protocol and resource   name.

Introduction To HTML:

1)HTML stands for Hyper Text Markup Language.
2)HTML is the standard markup language for creating Web pages. HTML describes the structure of a Web page. HTML consists of a series of elements. HTML elements tell the browser how to display the content.

History:

1)HTML was created by Sir Tim Berners-Lee in late 1991 but was not officially released.
2)HTML 1.0 was released in 1993 with the intention of sharing information that can be readable and accessible via web browsers. But not many of the developers were involved in creating websites. So the language was also not growing.
3)Then comes HTML 2.0, published in 1995, which contains all the features of HTML 1.0 along with a few additional features, which remained the standard markup language for designing and creating websites until January 1997 and refined various core features of HTML.
4)Then comes HTML 3.0, where Dave Raggett introduced a fresh paper or draft on HTML. It included improved new features of HTML, giving more powerful characteristics for webmasters in designing web pages. But these powerful features of the new HTML slowed down the browser in applying further improvements.
5)Then comes HTML 4.01, which is widely used and was a successful version of HTML before HTML 5.0, which is currently released and used worldwide. HTML 5 can be said for an extended version of HTML 4.01, which was published in the year 2012.
6)HTML 5 introduced so many semantic tags and input types.
